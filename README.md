## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Stack

- [Next.js 16](https://nextjs.org) — App Router
- [Supabase](https://supabase.com) — database & backend
- [Tailwind CSS v4](https://tailwindcss.com)
